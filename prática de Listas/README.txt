Informações:
toda implementação foi feita em C usando o vscode como IDE.
Para executar o programa basta abrir o terminal no vscode ir para a pasta onde está o executavel e digitar
./aplLuisEduardoSCMartins.exe.
Para compilar basta escrever gcc aplLuisEduardoSCMartins.c slListLuisEduardoSCMartins.o -o {nome do executavel},
para obter o slListLuisEduardoSCMartins.o é só digitar gcc -c slListLuisEduardoSCMartins.c no terminal, outra forma
de compilar o projeto é escrevendo gcc aplLuisEduardoSCMartins.c slListLuisEduardoSCMartins.c slListLuisEduardoSCMartins.h -o {nome do executavel}.
